# Flower Guts 実践BOOK - Digital Guide Book

腸内環境改善のための実践的なガイドブック。5つのカテゴリーで41の習慣をチェックリスト形式で管理できます。

## 🌸 特徴

- **5つのカテゴリー**: 食、からだ、自律神経、睡眠、トイレ
- **41の実践習慣**: 各カテゴリーの具体的な習慣をリストアップ
- **進捗管理**: チェックリスト形式で完了した項目を追跡
- **わかりやすいUI**: シンプルで使いやすいインターフェース
- **レスポンシブデザイン**: スマートフォンでも快適に利用可能

## 📖 カテゴリー

1. **食** - 食事に関する12の習慣
2. **からだ** - 姿勢・運動に関する9の習慣
3. **自律神経** - リラックス・ストレス対策の6の習慣
4. **睡眠** - 睡眠の質を高める9の習慣
5. **トイレ** - 排便習慣の5の習慣

## 🚀 使い方

1. [GitHub Pages](https://shihoyoga.github.io/flower-guts-book/) でアプリを開く
2. タブをクリックしてカテゴリーを切り替える
3. 実践した習慣をクリックしてチェック
4. 進捗状況を確認
